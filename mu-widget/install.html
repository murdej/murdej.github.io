MuWidget

A simple (java/type)script OOP component based framework for creating web UIs from simple widgets to the complete applications.

GitHub repository
NPM package
Try on CODEPEN
DISCORD channel

How to start using MuWidget

Option 1: NPM

You can simply install MuWidget via npm:

npm install mu-widget

Option 2: CDN

Or link script via CDN.

<script src="https://cdn.jsdelivr.net/gh/murdej/MuWidget2/dist/MuWidget-bundle.js"></script>

Option 3: Download

Another option is to download the bundle and link from your own server.

https://github.com/murdej/MuWidget2/blob/master/dist/MuWidget-bundle-nomodule.js
